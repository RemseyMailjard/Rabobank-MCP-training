What is it?

</> MCP

The RDS MCP Server connects GitHub Copilot or other MCP clients to the Rabobank Design System. It gives trusted access to RDS components, documentation, examples and best practices.

How it works

👨‍💻
Developer Asks in Copilot / IDE
→
🤖
GitHub Copilot MCP Client
→
🟧
RDS MCP Server Tools, APIs and approved guidance

Generates accurate, on-brand UI code and guidance based on Rabobank Design System standards.

Prompt:
Create a registration form using Rabobank Design System components.

</rds-form>

Access & prerequisites

  • GitHub Copilot account
  • Access to the required Rabobank AAD group
  • GitHub account registered with Rabobank email
  • GitHub organization SSO linked to Rabobank
  • VS Code version 1.116 or later
  • GitHub Copilot extension installed and signed in

Built for Rabobank developers

The RDS MCP Server helps developers create better digital experiences, faster — with the productivity of AI and the reliability of the Rabobank Design System.

Instead of asking an AI model to invent generic HTML, the developer gets guidance grounded in approved RDS components, documentation, examples and best practices.
